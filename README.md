# SocialMediaBolapelangi2 🌈

[![Vercel](https://img.shields.io/badge/Deployed%20on-Vercel-black)](https://vercel.com)

**SocialMediaBolapelangi2** adalah hub media sosial terlengkap yang menampilkan 8 platform populer dalam satu dashboard berwarna-warni.

## ✨ Fitur Lengkap
- 🟦 **Facebook** - Temukan teman & komunitas
- 🟪 **Instagram** - Visual stories & reels  
- 💚 **WhatsApp** - Pesan instan & panggilan video
- 🐦 **Twitter/X** - Trending topics & thread viral
- 💙 **Telegram** - Channel, bot & grup super
- 🔵 **Bluesky** - Social media desentralisasi
- ⚫ **TikTok** - Video pendek viral
- 🔴 **YouTube** - Streaming video & konten

## 🌈 Tampilan Unik
- Desain rainbow gradient yang eye-catching
- Animasi hover dengan border pelangi
- Floating rainbow button interaktif
- Preview chat real-time untuk WhatsApp, Telegram & Twitter
- Trending section dengan hashtag populer

## 🚀 Deployment
Website ini sudah siap di-deploy ke Vercel:

1. Upload ke GitHub
2. Import repository ke Vercel
3. Selesai! Website akan live

## 🛠️ Tech Stack
- HTML5
- CSS3 (Flexbox, Grid, Animations)
- JavaScript (ES6)
- Font Awesome Icons
- Google Fonts
